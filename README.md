# ai-knowledge-system

# 🧠 Personal AI System — личная AI-система

Оцифровал ключевые сферы жизни и встроил AI-инструменты 
в ежедневные процессы: работа, обучение, здоровье, семья.

---

## Из чего состоит система

| Инструмент | Роль |
|-----------|------|
| Obsidian | База знаний и документация |
| Claude Projects | AI-ассистенты под задачи |
| Make.com | Автоматизация процессов (в разработке) |

---

## Obsidian — база знаний

Структурированная база данных по всем сферам жизни:

- 📁 Знания (веб-разработка, ИИ, реабилитация, языки)
- 📁 Проекты (личный ассистент, массаж, digital-агентство)
- 📁 Профиль (планирование, здоровье, семья)

![Граф знаний Obsidian](obsidian-graph.png)

---

## Claude Projects — AI-ассистенты

Структура повторяет Obsidian — каждый проект имеет 
своего ассистента с кастомным контекстом:

- **Личный ассистент** — задачи, клиенты, автоматизация
- **Массаж и реабилитация** — работа с пациентами
- **Обучение** — веб-разработка, ИИ, саморазвитие

---

## Пример промпта

**ПРОМПТ: Карта пациента из выписки**

На основе первичного анамнеза, выписок и данных 
исследований (МРТ, УЗИ) формирует структурированную 
карту пациента для работы с конкретным диагнозом.

![Пример промпта](prompt-example.png)

---

## Результат

Системный и структурированный подход к работе и обучению.
Каждая задача решается с AI-поддержкой и сохраняется 
в базе знаний для повторного использования.
